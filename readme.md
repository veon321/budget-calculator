# Budget Calculator

A simple budget calculator built with pure JavaScript, HTML, and CSS. The application allows users to manage their daily expenses, set a total budget, and track the percentage of the budget used.

## 🚀 Live Demo

Check out the live application here: [Budget Calculator Live](https://veon321.github.io/budget-calculator/)

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Responsive Design)
- JavaScript (Vanilla JS, DOM Manipulation, Array Methods)
